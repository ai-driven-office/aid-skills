# Ocean Digital

A cool, trustworthy theme using Spindle's Expressive Blue palette.

## Color Palette

- **Deep Ocean**: `#203957` - Primary accent
- **Sky Light**: `#e3f4fa` - Soft backgrounds
- **Digital Blue**: `#4da2d6` - Interactive highlights
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Technology presentations, digital services, trust-building content, professional consulting.
