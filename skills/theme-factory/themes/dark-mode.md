# Dark Mode

A sophisticated dark theme for premium, high-impact presentations.

## Color Palette

- **Deep Black**: `#08121a` - Primary background
- **Dark Surface**: `#141e25` - Card backgrounds
- **Bright Green**: `#41ad4f` - Accent highlights
- **White**: `#ffffff` - Primary text
- **Gray 30**: `#b5b8ba` - Secondary text

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Executive presentations, product reveals, tech showcases, evening events, premium content.
