# Fresh Lime

An energetic, vibrant theme using the secondary Yellow Green from Spindle.

## Color Palette

- **Yellow Green**: `#82be28` - Primary accent
- **Lime Light**: `#f0f7e6` - Backgrounds
- **Deep Lime**: `#477d00` - Strong emphasis
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Marketing materials, product launches, innovation presentations, startup pitches.
