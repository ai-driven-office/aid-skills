# Deep Green

A sophisticated, professional theme using the darkest Primary Green tones.

## Color Palette

- **Deep Green**: `#0f5c1f` - Primary accent
- **Green Mist**: `#c6e5c9` - Secondary backgrounds
- **Forest**: `#298737` - Interactive elements
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Executive presentations, financial reports, sustainability content, premium communications.
