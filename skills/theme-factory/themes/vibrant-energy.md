# Vibrant Energy

A bold, energetic theme using Spindle's Expressive Pink palette.

## Color Palette

- **Deep Rose**: `#630f33` - Primary accent
- **Blush Light**: `#fce4e9` - Soft backgrounds
- **Vibrant Pink**: `#e02c53` - Highlights
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Marketing campaigns, event promotions, youth-focused content, dynamic presentations.
