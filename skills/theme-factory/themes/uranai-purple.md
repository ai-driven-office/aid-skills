# Uranai Purple

The official theme for Uranai (fortune-telling) sub-brand with mystical purple tones.

## Color Palette

- **Navy Purple**: `#391e66` - Primary accent
- **Lavender Mist**: `#eee8f4` - Soft backgrounds
- **Uranai Purple**: `#743faa` - Highlights
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Uranai brand materials, entertainment content, creative presentations, lifestyle services.
