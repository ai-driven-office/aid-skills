# Neutral Pro

A clean, professional theme with minimal color accents for maximum versatility.

## Color Palette

- **Charcoal**: `#08121a` - Primary text and headers
- **Light Gray**: `#f5f6f6` - Subtle backgrounds
- **Ameba Green**: `#2d8c3c` - Accent highlights
- **Gray 60**: `#686e73` - Secondary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Technical documentation, data presentations, reports, corporate templates.
