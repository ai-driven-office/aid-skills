# Warm Sunset

A warm, inviting theme using Spindle's Expressive Orange palette.

## Color Palette

- **Burnt Orange**: `#8a2701` - Primary accent
- **Peach Cream**: `#f6e7e5` - Soft backgrounds
- **Vibrant Orange**: `#de4d14` - Highlights
- **Gray 100**: `#08121a` - Primary text
- **White**: `#ffffff` - Slide backgrounds

## Typography

- **Headers**: Meiryo (Bold)
- **Body Text**: Meiryo

## Best Used For

Creative presentations, entertainment content, lifestyle brands, warm storytelling.
