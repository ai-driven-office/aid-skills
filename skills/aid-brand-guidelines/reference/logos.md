# AID Logo Specifications

## Logo Files

| File | Format | Use Case |
|------|--------|----------|
| `rgb_Logomark.png` | PNG | Digital — full logomark |
| `cmyk_Logomark.png` | PNG | Print — full logomark |
| `rgb_Symbol Mark.png` | PNG | Digital — symbol only |
| `cmyk_Symbol Mark.png` | PNG | Print — symbol only |
| `ai_driven_logo_rgb.ai` | AI | Digital vector (recommended) |
| `ai_driven_logo_cmyk.ai` | AI | Print vector |

## Logo Forms

| Form | Description | Use |
|------|-------------|-----|
| Full Logomark | "AI := Driven" (text + symbol) | Primary usage |
| Symbol Mark | `:=` glyph only | Icons, favicons, avatars |
| Template | "AI := ___" | Campaigns, events |

## Logomark Construction

### Layout

- Two-line stack: top "AI :=" / bottom "Driven"
- Left-aligned
- "AI" is text; `:=` is the symbol mark, aligned to the same height as "AI"

### Typography

- Typeface: Bold geometric sans-serif (extra-black weight)
- "AI" — uppercase, black weight
- "Driven" — title case (capital D), bold weight
- "Driven" has rounded terminals (soft ends on r, i, v, e, n)
- The dot on "Driven"'s "i" is a small solid circle, consistent with the brand dot language
- Contrast: "AI"'s hard straight lines vs. "Driven"'s organic curves — implying human-AI collaboration

## Symbol Mark `:=` Construction

### Colon (`:`) — Two Circles

- **Top circle**: Red/Magenta (`#FF0413`), solid fill
- **Bottom circle**: Blue (`#3370FE`), solid fill

### Colon Connection Trail

- Dithering/stipple (particle) texture between and below the two circles
- Color transitions Red → Purple → Blue along the trail
- Particles scatter and fade to background color toward the bottom
- Represents "Driven" — energy radiating and dissipating

### Equals (`=`) — Two Horizontal Bars

- **Top bar**: Pink/Magenta → Red gradient (left → right)
- **Bottom bar**: Indigo/Purple → Magenta/Violet gradient (left → right)
- Both bars are sharp rectangles — no border-radius

## Dark Background Variant

- Symbol mark works on dark (black) backgrounds
- Gradient and dithering textures are enhanced against dark backgrounds
- Text switches to white on dark backgrounds
